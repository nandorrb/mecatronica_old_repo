¿Como ahorrar presupuesto en una empresa?

Sofware Privativo VS Sofware Libre

Es comun en el Perú usar Windows 7, o incluso Windows XP, así como MS SQL server, MS Office y otros programas de Microsoft, todos privativos.

Pero ¿Que es software privativo? es un tipo de software por el cual tu debes pagar una licencia para poder usarlo.

¿Entonces existe sofware por el que no se paga? ¡Debe ser malo! : El sentido comun y los prejuicios cuestan miles de dolares a una empresa, el software libre no es malo, es un camino al ahorro y reduccion de costos.

El software libre tiene las mismas funciones, con algunas ventajas y desventajas. Ventajas como la constante actualización gratuita, lo cual permite tener siempre lo último en tecnologia a costo cero, así como desventajas como que requiere una capacitación extra de personal para usarlo.

¿Que ofrece Mecatronica.pe a respecto?

Ofrecemos Sentar las bases técnicas y metodológicas para llevar a cabo el proceso de migración de empresas hacia el software libre. Así como servicios de asesoría, capacitación y mantenimiento.

La principal razon por la cual no se usa Software libre, es que no existen expertos que permitan a las empresas favorecer la migración. Eso cambió con nosotros.
